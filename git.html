Git

  • git init - создание нового локального репозитория.
  • git status - перечисляет все новые или измененные файлы, которые нуждаются в фиксации.
  • git add [файл] - индексирует указанный файл для последующего коммита.
  • git reset [файл] - отменяет индексацию указанного файла, сохраняя его содержимое.
  • git commit -m "[сообщение с описанием]" - фиксирует проиндексированные изменения и сохраняет их в историю версий.
